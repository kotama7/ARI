\section{Runtime LLM Prompts}\label{sec:prompts}

This appendix lists the LLM prompts that ARI uses at runtime, reproduced
verbatim from \texttt{ari-core/ari/prompts/} at v0.8.0. Each listing is
byte-for-byte identical to the source on disk. Because the exact bytes are
what reach the model, the prompts are reproduced as-is in every language
version of this report and are not translated.
