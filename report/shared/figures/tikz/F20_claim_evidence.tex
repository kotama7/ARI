% F20: ARI's execution-grounded realisation of the Story2Proposal contract.
% A run-level METRIC CONTRACT declares the falsifiable claims the eventual
% paper must support, each naming the exact measurement names that count as
% evidence. The contract is minted once and reused verbatim thereafter (a
% mid-run regeneration would silently rename the evidence vocabulary).
% Exploration nodes emit measurements under those names; an advisory
% point-of-emission check warns about uncovered claims and surfaces
% suggestion-only lexical name hints so the agent can re-emit under the
% required names (nothing is auto-bound; the gate never sees this logic).
% Manuscript claims --- numeric assertions and figure references --- are
% bound by stable node/result/figure ids to the recorded evidence, and a
% deterministic claim--evidence gate (no LLM) re-derives each reported
% number within tolerance: the data-fidelity axis blocks in strict mode,
% the reasoning and visual-coherence axes are advisory.
% style.tikzstyles is loaded by preamble.tex / the standalone wrapper; the
