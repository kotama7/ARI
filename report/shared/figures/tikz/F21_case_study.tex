% F21: the gate-verified sample paper as a worked example of the contract.
% The run-level metric contract declares twelve falsifiable claims: nine are
% INDEPENDENT PROBES (correctness, store policy, gather/locality, page
% footprint, bandwidth) that a node measures directly, and three form a
% COMPUTED-EVIDENCE CHAIN (fit the performance-model parameters, cross-validate
% the fit on held-out probes, then select a configuration from the fitted
% model). The chain is reachable only by LINEAGE CLOSURE: a child node inherits
% its parent probe node's working directory and computes the chain evidence
% from the inherited measurements instead of re-running probes. The
% deterministic claim--evidence gate then finds every claim grounded (zero
% errors) and the paper is finalised.
