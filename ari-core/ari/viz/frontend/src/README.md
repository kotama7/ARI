# frontend/src

- `main.tsx` тАФ ReactDOM entry point with a top-level error boundary.
- `vite-env.d.ts` тАФ Vite client type declarations.

    - `Card.tsx` тАФ bordered content container.
    - `index.ts` тАФ barrel re-exports.

    - `StatusBadge.tsx` тАФ maps run status to a colored `Badge`.
  - `Experiments/` тАФ experiments page (lists experiment/checkpoint runs).
    - `README.md` тАФ Experiments index.
    - `ExperimentsPage.tsx` тАФ experiments list view.
    - `index.ts` тАФ barrel re-export.
  - `Home/` тАФ home/overview landing page.
    - `README.md` тАФ Home index.
    - `HomePage.tsx` тАФ home/overview view.
    - `index.ts` тАФ barrel re-export.
  - `Idea/` тАФ idea page (research idea/hypothesis).
    - `README.md` тАФ Idea index.
    - `IdeaPage.tsx` тАФ idea view.
    - `index.ts` тАФ barrel re-export.
  - `Layout/` тАФ app shell (page frame + nav sidebar).
    - `README.md` тАФ Layout index.

    - `monitorSections.tsx` тАФ metric helper + Experiment-Configuration card (extracted from MonitorPage in req 15).

    - `results/` тАФ rubric-scored results view.
