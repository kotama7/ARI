# ari

Core engine package for ARI. Each sub-package carries its own `README.md`
(thin entry point) and an authoritative module docstring in `__init__.py`.

## Contents

- `README.md` ‚Äî this file.
- `__init__.py` ‚Äî package marker.
- `_deprecation.py` ‚Äî `DeprecationWarning` helpers for v0.5‚Üív1.0 legacy paths/aliases.
- `checkpoint.py` ‚Äî checkpoint JSON I/O (`tree.json` / `nodes_tree.json` / `results.json`).
- `cli_ear.py` ‚Äî `ari ear ‚Ä¶` curation / publish / promote / status CLI surface.
- `container.py` ‚Äî unified container runtime abstraction (Docker / Singularity / bare subprocess).
- `core.py` ‚Äî generic ARI runtime builder (zero domain-specific code; `cli` calls into it).
- `cost_tracker.py` ‚Äî per-call logs + per-experiment cost summaries.
- `env_detect.py` ‚Äî detect schedulers, container runtimes, and HPC resources.

    - `latex.py` ‚Äî deterministic LaTeX section + numeric-token parsing (coverage fallback; mirrors ari-skill-paper/src/claim_links.py).
    - `numeric.py` ‚Äî formula registry + `recompute` + `within_tolerance` (Phase B2; mirrored in ari-skill-transform/src/claims.py).
    - `policy.py` ‚Äî `claim_gate_policy` loader (defaults ‚Üí arg ‚Üí `claim_gate_policy.json` ‚Üí env `ARI_CLAIM_GATE_MODE`).
