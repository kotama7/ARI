# ari-skill-paper/templates

LaTeX paper templates, one subdirectory per target venue, consumed by
`write_paper_iterative` when emitting a full paper. Each subdir holds
`main.tex` (the venue-styled document skeleton) and `refs.bib` (bibliography).
